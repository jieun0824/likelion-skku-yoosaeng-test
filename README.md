### 사용 스택

- Next.js + typescript
- tailwind css
- SWR

- 이미지 생성 - chatGPT 생성형 ai 사용

### 이용 방법

유생 테스트를 통해 전생에 내가 어떤 유생 유형에 해당했는지를 확인해볼 수 있습니다.
결과창은 카카오톡 공유하기 기능을 통해 공유가 가능합니다.
